# Integrated Impact Assessment

This is a web-based prototype for a Sport Wales project.

## Getting started
1. Clone this repository
2. Install necessary Node packages
```bash
cd app
```
```bash
npm install
```
3. Start dev server
```bash
npm run dev
```

## Technology stack
- React (& Vite)
- Tailwind (via CDN)
